## Demo扼要:

- 整体基于antdesignPro v4 搭配Umi3.0
- 运用React Hooks
- antd新组件Protable 运用
- Umi3.0融入了Dva 不再单独引用dva
- 配置了request （返回及请求拦截）
- 拎包入住（添加页面即可）



## 参考文档:
- Umi3.0: https://umijs.org/
- Dva: https://dvajs.com/
- Ant Design Pro: https://preview.pro.ant.design/dashboard/analysis
- Protable: https://procomponents.ant.design/components/table
